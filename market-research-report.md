**작성일**: 2026년 3월 31일  
**대상 앱**: The Royal Closet (royal-dress-up)  
**기술 스택**: Next.js 16, React 19, Fabric.js, Tailwind CSS 4

## 1. 시장 개요

### 1-1. 가상 피팅 / 트라이온 시장

| 항목 | 수치 |
|---|---|
| 2024년 시장 규모 | USD 9.59B ~ 11.38B |
| 2025년 시장 규모 | USD 12.09B |
| 2026년 시장 규모 | USD 8.27B (가상 피팅룸 세그먼트) |
| 2030년 전망 | USD 46.42B |
| CAGR (2024-2030) | **26.4%** |

- 의류/패션 세그먼트가 2025년 전체 가상 트라이온 플랫폼 매출의 **39.6%** 차지
- 북미 시장이 2025년 USD 2.64B, 2026년 USD 3.17B로 선도
- 쇼핑객의 **55%**가 AR 기능을 사용하고, 소매업체의 **46%**가 AI 기반 개인화 시스템을 배포

### 1-2. 드레스업 게임 시장

| 항목 | 수치 |
|---|---|
| 2024년 시장 규모 | USD 13.75B |
| 2025년 시장 규모 | USD 4.2B (게임 세그먼트) |
| 2032년 전망 | USD 26.5B |
| CAGR | **5.0% ~ 8.6%** |

- 모바일 게임 세그먼트가 **58.5%** 점유율로 시장 지배
- 상위 드레스업 게임의 월간 활성 사용자(MAU) 유지율: 34.2% (2025년, 2022년 28.6% 대비 개선)
- 북미 신규 설치의 **35%** 이상이 소셜 미디어를 통해 유입 (2025년)

### 1-3. 타겟 인구통계

- **핵심 타겟**: 18~35세 여성, 패션/로열 패밀리에 관심
- **확장 타겟**: 10대 청소년 (드레스업 게임), 패션 전문가, 로열 패밀리 팬덤
- **지역**: 영미권 (UK, US, AU) 중심, 한국/일본 등 아시아 확장 가능

## 3. 시장 기회

### 3-1. 블루오션 영역

1. **로열 패밀리 특화 니치**: 순수하게 로열 패밀리 패션에 특화된 경쟁 앱이 거의 없음
2. **웹 기반 접근성**: 대부분의 드레스업 게임이 모바일 네이티브 앱인 반면, The Royal Closet은 웹 앱으로 설치 없이 접근 가능
3. **역사적 패션 교육**: Grace Kelly, Meghan Markle 등 실제 로열 패션 히스토리를 결합한 에듀테인먼트
4. **소셜 미디어 바이럴**: 로열 패션 "나도 입어보기" 콘텐츠는 SNS에서 높은 공유율 예상

### 3-2. 성장 가능 방향

| 기회 | 설명 | 우선순위 |
|---|---|---|
| **AI 가상 트라이온 고도화** | 사용자 사진에 로열 드레스를 합성하는 AI 기능 | 높음 |
| **시즌/이벤트 컬렉션** | 로열 웨딩, 대관식, 가든파티 등 이벤트 연동 | 높음 |
| **소셜 공유 강화** | "나의 로열 룩" 결과 이미지 SNS 공유 최적화 | 높음 |
| **Bridgerton 등 IP 확장** | 넷플릭스 시대극 팬덤 타겟 | 중간 |
| **커뮤니티 기능** | 사용자 스타일링 투표, 랭킹, 챌린지 | 중간 |
| **다국어 지원** | 한국, 일본, 유럽 시장 진출 | 중간 |

## 6. 핵심 권장 사항

### 단기 (1~3개월)

1. **AI 가상 트라이온 MVP**: 사용자 사진 + 로열 드레스 합성 기능 프로토타입
2. **SNS 공유 최적화**: OG 이미지, 인스타그램/틱톡 최적화 결과 이미지
3. **SEO/콘텐츠 마케팅**: 로열 패션 히스토리 블로그, "What would you wear to..." 콘텐츠

### 중기 (3~6개월)

4. **구독 모델 도입**: 프리미엄 콘텐츠 잠금 + 구독 결제
5. **이벤트 시스템**: 실제 로열 이벤트(웨딩, 생일 등)와 연동된 한정 컬렉션
6. **커뮤니티 기능**: 사용자 스타일링 갤러리, 투표 시스템

### 장기 (6~12개월)

7. **모바일 앱 (PWA/네이티브)**: 모바일 사용자 확보
8. **IP 파트너십**: Bridgerton, The Crown 등 콘텐츠 라이센스
9. **글로벌 확장**: 다국어 지원, 각국 왕실 컬렉션 추가

### 10-2. 왕실 초상권 및 퍼블리시티권 (가장 중요)

#### 영국 왕실 이미지 사용 규정

- **Royal Images** 기관에 따르면, 왕실 구성원의 이미지는 **서면 동의 없이 상업적 목적으로 사용할 수 없음**
- 왕실 구성원 이미지 근처에 **회사명이나 제품 광고를 배치하는 것도 금지**
- 모든 사용 신청은 개별 심사되며, 왕실에 **부적절하거나 해로운 경우 허가 거부**
- 승인은 Buckingham Palace를 포함한 여러 관계자의 동의가 필요

#### 미국 퍼블리시티권 (Right of Publicity)

- 미국 34개 주에서 **개인의 이름, 이미지, 초상의 상업적 사용을 규제**하는 별도 법률 존재
- Meghan Markle (현재 미국 거주)의 경우 미국법 적용 가능
- 캘리포니아 Civil Code §3344: 상업적 목적의 초상 무단 사용 시 **최소 $750 법정 손해배상 + 실제 손해액**
- 사망 후에도 보호 (Grace Kelly의 경우 상속인이 권리 보유 가능)

### 10-3. 패션 디자인 저작권

- 미국/영국 법상 의류는 **"기능적 물품(useful articles)"**으로 분류 → 저작권 보호 제한

### 10-5. 사용자 데이터 및 프라이버시

#### GDPR (EU 일반 데이터 보호 규정)

- 가상 트라이온 시 사용자 사진 업로드 → **생체 데이터(biometric data)**로 분류 가능

- **명시적 동의(explicit consent)** 및 강화된 보안 조치 필수
- 위반 시 벌금: **전 세계 연매출의 4% 또는 €2,000만 중 높은 금액**
- 2025년 12월 기준 유럽 GDPR 벌금 누적: **€67억+** (2,679건)

#### 2026년 주요 프라이버시 규제 변화

| 규제 | 영향 |

| **EU Data Act** | 사용자 생성 데이터의 접근·이동 권리 강화 |
| **미국 주별 프라이버시법** 확대 | 캘리포니아(CCPA/CPRA), 텍사스, 오레곤 등 생체정보 규제 강화 |
| **아동 온라인 보호** | 10대 타겟 시 COPPA(미국)/Age Appropriate Design Code(영국) 준수 필요 |

#### 핵심 기술 원칙: Privacy by Design

| 원칙 | 구현 방법 | 우선순위 |
|---|---|---|
| **클라이언트 사이드 처리** | 사진을 서버로 전송하지 않고 브라우저(Fabric.js)에서만 처리 | 🔴 필수 |
| **데이터 최소화** | 트라이온에 필요한 최소 데이터만 처리, 메타데이터(EXIF) 자동 제거 | 🔴 필수 |
| **자동 삭제** | 세션 종료/브라우저 닫기 시 모든 사진 데이터 자동 삭제 | 🔴 필수 |
| **대안 제공** | 사진 업로드 거부 시 아바타/마네킹 기반 트라이온 제공 | 🔴 필수 (GDPR 요구) |
| **동의 철회** | 언제든 동의 철회 가능 + 즉시 데이터 삭제 버튼 | 🟡 권장 |
| **투명성** | 어떤 데이터가 어떻게 처리되는지 평문으로 설명 | 🟡 권장 |

#### 프라이버시 정책 필수 포함 항목

1. **수집하는 데이터 유형**: 사진, 쿠키, 사용 로그 등
2. **데이터 처리 목적**: 가상 트라이온 기능 제공
3. **법적 근거**: 사용자 동의 (GDPR Art.6(1)(a) + Art.9(2)(a))
4. **데이터 보관 기간**: 세션 중에만 / 저장하지 않음
5. **제3자 공유 여부**: 공유하지 않음 (또는 공유 대상 명시)
6. **사용자 권리**: 접근, 수정, 삭제, 이동, 처리 제한, 반대 권리
7. **쿠키 정책**: 사용하는 쿠키 유형과 목적
8. **연락처**: 데이터 보호 관련 문의처
